About the App

FRACTION-matiks is an educational web-based application developed to help users learn and practice basic fraction concepts. The application contains three sections:

  • Lessons: Users can access different lessons on fractions.
  • Games: Users can access games to improve their fraction-related skills.
  • Quiz: Users can access quizzes to assess their learning. Teachers can also create custom quizzes for students, which will appear in this section.
